# Sound Effects Library

## Built-in SFX

These come from the @remotion/sfx package and are available by ID without
local files. The manifest at `assets/sfx/manifest.json` catalogs all available
sounds.

To add a local custom SFX file, drop it here and add an entry to the manifest
with `"source": "local"`.

## Royalty-Free Sources

- [freesound.org](https://freesound.org/) — community sound library (check CC license per clip)
- [kenney.nl](https://kenney.nl/assets?category=Audio) — game audio assets, CC0
- [pixabay.com/sound-effects](https://pixabay.com/sound-effects/) — royalty-free SFX
- [mixkit.co/free-sound-effects](https://mixkit.co/free-sound-effects/) — free, no attribution

## File Format

WAV or MP3. Keep short (< 2 seconds) for cut-point and emphasis SFX.
